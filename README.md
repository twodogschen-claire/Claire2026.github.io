# 🎯 Nexus Core - AI 学习规划系统 Demo

[![Live Demo](https://img.shields.io/badge/Live_Demo-在线体验-success.svg)](https://Claire2026.github.io/)

> **本项目为“AI 学习规划工具”的网页端单页原型 Demo，旨在展示教育科学理论与大模型结合的智能排期产品逻辑。**

## 💡 项目简介
“Nexus Core” 是一位兼具科学专业性与人文关怀的全能学习规划助手。传统工具仅能做简单的任务堆砌，而本项目基于前沿教育学终身理论，设计了**五大专家角色协同机制**：
- 🧠 **学习科学家**：拆解里程碑与知识路径
- 🗓️ **时间安排规划师**：精准匹配精力周期与碎片时间
- 🔍 **信息检索员**：甄别筛选优质资料
- 🎓 **资深教师**：沉浸式互动引导教学
- 🌿 **身心健康咨询师**：动态情绪识别与干预

## ✨ 核心功能演示
- **🚀 智能向导配置**：通过极简的卡片化交互（Onboarding Wizard），30秒内收集用户的核心目标、基础痛点、备考周期与时间资源。
- **📊 全局与微观双重视图**：
  - **本周全局概览**：明确宏观重心、习惯养成指标与容错预案。
  - **每日详细导览**：精确到半小时级的颗粒度排期，智能填充碎片化时间。
- **⚡ 动态智能路由**：底层预置策略逻辑，能够根据用户输入的“雅思/考研/期末”及“全职/兼职”等变量，动态生成差异化的优质学习路径。

## 🛠️ 技术栈
- **UI 规范**：卡片式极简现代设计（参考 Notion/Flowith），注重留白与排版。
- **前端实现**：纯 `HTML5` + `CSS3` + `Vanilla JavaScript`，零外部依赖，极速加载。
- **部署方案**：通过 `GitHub Pages` 实现自动化云端部署上线。
